# WASM Demo

This folder contains the browser-side files for the LLVM WebAssembly extra-credit path.

## Files

- `index.html` - feature showcase page for the exported wasm functions
- `runtime.js` - loads `demo.wasm`, provides a tiny `malloc` import, and calls the exported functions
- `demo.o` - generated by `build_wasm_demo.sh`
- `demo.wasm` - generated by `build_wasm_demo.sh` when `wasm-ld` is available

## Build

From the project root:

```bash
bash build_wasm_demo.sh
```

The script:

1. compiles the Java sources
2. generates LLVM IR from `tests/compiler/wasm_export_demo.pas`
3. uses `llc` to create `wasm/demo.o`
4. links `wasm/demo.wasm` if `wasm-ld` is installed

## Showcase Exports

The current demo exports:

- `AddOne`
- `MaxValue`
- `SumToN`
- `CounterDemo`

These cover arithmetic, branching, loops, and a class-backed method path.

## Serve

Because browsers fetch `.wasm` over HTTP, serve the project directory locally:

```bash
python3 -m http.server 8000
```

Then open:

`http://127.0.0.1:8000/wasm/`

If the page says module load failed, check whether `wasm/demo.wasm` was generated.
